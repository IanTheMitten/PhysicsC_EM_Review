# AP Physics C: Electricity & Magnetism — Study Guide

A visual HTML study guide covering the entire AP Physics C: E&M curriculum. Includes all 6 College Board units (Units 8-13) organized into 5 focused chapters with concepts, equations, practice problems, and exam strategy.

## Quick Start

```bash
cd "path/to/physicsEM"
python3 -m http.server 8080
```

Then open **http://localhost:8080** in your browser.

Or just double-click `index.html` to open it directly.

## What's Inside

- **5 Study Units** — Electrostatics, Circuits, Magnetism, Electromagnetism, Maxwell's Equations
- **~85 Practice Questions** — With step-by-step solutions
- **59 Key Equations** — Rendered with LaTeX (MathJax)
- **Exam Strategy** — Time management, FRQ tips, lab scenarios, common mistakes
- **Practice Materials Directory** — Links to all PDF practice tests and notes
- **Dark Mode** — Toggle with the moon icon or press `T`
- **Progress Tracking** — Checkboxes saved to your browser
- **Keyboard Shortcuts** — Arrow keys navigate, 1-5 jump to units, `/` to search
